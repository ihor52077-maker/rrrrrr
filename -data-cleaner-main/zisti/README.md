# Очиститель данных - Расширенный инструмент обработки данных

Обработка результатов DorkStrike PRO с классификацией и очисткой данных.

## Запуск

```bash
python data_cleaner/data_cleaner.py
Фи
✅ 11-этапный конвейер обработки
✅ Классификация по важности (КРИТИЧЕСКИЕ, ПОЛЕЗНЫЕ, СЛАБЫЕ, МУСОРНЫЕ)
✅ Быстрая обработка больших файлов
✅ Графический интерфейс на Tkinter (как DorkStrike PRO)
✅ Статистика без процентов
✅ Поддержка ZIP, JSON, TXT, CSV
Выходные файлы
output/critical.txt- Токены, API-ключи, приватные ключи, электронная почта:пароль
output/useful.txt- логин:пароль, куки, конфиги
output/weak.txt- Электронные письма, имена пользователей, конечные точки
output/trash.txt- Нерелевантные данные
output/stats.json- Статистика обработки
